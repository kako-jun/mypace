# リアクション（ステラ）

投稿に反応する方法です。

## ステラとは

MY PACEでは「いいね」の代わりに「ステラ」（⭐️）を使います。

- 1つの投稿に最大10個まで
- 5種類の色を組み合わせ可能
- 連打OK！気持ちを込めて送りましょう

## 送り方

1. 投稿の⭐️アイコンをクリック
2. カラーピッカーが表示される
3. 送りたい色の星をクリック（クリックごとに1つ追加）
4. 複数の色を組み合わせ可能（合計10個まで）

## カラーステラ

ステラには5種類の色があります。

| 色 | コスト | 説明 |
|----|--------|------|
| 🌟 イエロー | 無料 | 通常のステラ |
| 💚 グリーン | 1 sat | ちょっとしたお礼に |
| ❤️ レッド | 10 sats | 気に入った投稿に |
| 💙 ブルー | 100 sats | とても良い投稿に |
| 💜 パープル | 1000 sats | 最高の投稿に！ |

### 複数色の組み合わせ

1つの投稿に複数の色を送れます:

- イエロー5個 + グリーン2個 = 合計7個
- ブルー1個 + レッド3個 = 合計4個

### カラーステラを送るには

1. ウォレットを接続（設定→ウォレット）
2. ステラボタンをクリック
3. カラーピッカーから送りたい色をクリック
4. 連打で同じ色を追加可能
5. 別の色をクリックすればその色も追加

### 注意事項

- イエローステラは常に取り消し可能です（カラーステラがあっても）
- カラーステラ（有料）は支払い済みなので取り消しできません（残ります）
- 相手がLightningアドレス（lud16）を設定している必要があります

## 投稿カードでの表示

リプライやリポストと同じスタイルで、色別のカウントが表示されます:

```
🌟  💚  💙
 3   2   1
```

- 各色の星の下に数字が表示されます（ステラがない場合は「0」）
- **星をクリック**: カラーピッカーが表示されます（ステラを追加）

## リアクターを見る

誰がどの色のステラを送ったか確認できます。

1. 星をクリックしてカラーピッカーを開く
2. 各星の下にある⋮アイコンをクリック
3. その色のリアクター一覧がポップアップ表示される

リアクター一覧では、その色を付けたユーザーとカウントが表示されます。

自分の投稿でも同じ操作ができます（カラーピッカー内のステラ追加ボタンは無効）。

## 返信

投稿に返信するには:

1. 💬アイコンをクリック
2. 返信を入力
3. 「投稿」をクリック

## リポスト

投稿を自分のタイムラインに共有するには:

1. 🔁アイコンをクリック
2. 確認して「リポスト」

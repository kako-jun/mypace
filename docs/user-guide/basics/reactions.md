# リアクション（ステラ）

投稿に反応する方法です。

## ステラとは

MY PACEでは「いいね」の代わりに「ステラ」（⭐️）を使います。

- 1つの投稿に最大10個まで
- 5種類の色を組み合わせ可能
- 連打OK！気持ちを込めて送りましょう

## 送り方

1. 投稿の⭐️アイコンをクリック
2. カラーピッカーが表示される
3. 送りたい色の星をクリック（クリックごとに1つ追加）
4. 複数の色を組み合わせ可能（合計10個まで）

## カラーステラ

ステラには5種類の色があります。

| 色          | コスト    | 説明               |
| ----------- | --------- | ------------------ |
| 🌟 イエロー | 無料      | 通常のステラ       |
| 💚 グリーン | 1 sat     | ちょっとしたお礼に |
| ❤️ レッド   | 10 sats   | 気に入った投稿に   |
| 💙 ブルー   | 100 sats  | とても良い投稿に   |
| 💜 パープル | 1000 sats | 最高の投稿に！     |

### カラーステラの獲得方法

グリーンステラ以上は以下の方法で獲得できます:

1. **他の人から受け取る**
   - 誰かがあなたの投稿にカラーステラを送ると、それがあなたのインベントリに追加されます
   - 受け取ったステラは自由に使えます（別の投稿に送信可能）

2. **Supernova（目標達成）の報酬**
   - 特定の条件を達成すると、報酬としてカラーステラがもらえます
   - 詳しくは[インベントリページ](../../features/inventory.md)を確認してください

イエローステラは無限に使えるため、インベントリには追加されません。

### 複数色の組み合わせ

1つの投稿に複数の色を送れます:

- イエロー5個 + グリーン2個 = 合計7個
- ブルー1個 + レッド3個 = 合計4個

### カラーステラを送るには

1. ウォレットを接続（設定→ウォレット）
2. ステラボタンをクリック
3. カラーピッカーから送りたい色をクリック
4. 連打で同じ色を追加可能
5. 別の色をクリックすればその色も追加

### 取り消し方法

すべての色のステラを取り消すことができます:

1. ステラボタンをクリックしてカラーピッカーを開く
2. 取り消したい色の⋮アイコンをクリック
3. リアクター一覧で自分の「Remove」をクリック

カラーステラを取り消すと、インベントリに返却されます（リファンド）。

**注意**:

- 受け取った相手が既にそのステラを使用している場合、相手の残高がマイナスになることはありません（0で保護されます）
- 受け取った人が更に別の人にステラを送っていても、あなたが取り消せるのは直接送った相手との関係のみです

### 注意事項

- 相手がLightningアドレス（lud16）を設定している必要があります

## 投稿カードでの表示

リプライやリポストと同じスタイルで、色別のカウントが表示されます:

```
🌟  💚  💙
 3   2   1
```

- イエローの星は常に表示されます
- その他の色は1つ以上付けられた場合のみ表示されます
- どの色の星をクリックしても同じカラーピッカーが開きます
- マウスを乗せると、すべての星がまとめて拡大されます

## リアクターを見る

誰がどの色のステラを送ったか確認できます。

1. 星をクリックしてカラーピッカーを開く
2. 各星の下にある⋮アイコンをクリック
3. その色のリアクター一覧がポップアップ表示される

リアクター一覧では、その色を付けたユーザーとカウントが表示されます。

自分の投稿でも同じ操作ができます（カラーピッカー内のステラ追加ボタンは無効）。

## 返信

投稿に返信するには:

1. 💬アイコンをクリック
2. 返信を入力
3. 「投稿」をクリック

## リポスト

投稿を自分のタイムラインに共有するには:

1. 🔁アイコンをクリック
2. 確認して「リポスト」

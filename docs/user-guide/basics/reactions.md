# リアクション（ステラ）

投稿に反応する方法です。

## ステラとは

MY PACEでは「いいね」の代わりに「ステラ」（⭐️）を使います。

- 1つの投稿に最大10個まで
- 同じ投稿に何度もステラを送れます

## 送り方

1. 投稿の⭐️アイコンをクリック
2. クリックごとに1つずつ追加（最大10個）

## リアクターを見る

誰がステラを送ったか確認できます。

- ⭐️の数をクリック
- または長押し

リアクター一覧が表示されます。

## 返信

投稿に返信するには:

1. 💬アイコンをクリック
2. 返信を入力
3. 「投稿」をクリック

## リポスト

投稿を自分のタイムラインに共有するには:

1. 🔁アイコンをクリック
2. 確認して「リポスト」

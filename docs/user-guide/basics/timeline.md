# タイムライン

投稿を閲覧する画面の使い方です。

## 基本

タイムラインにはユーザーの投稿が新しい順に表示されます。

### 投稿カード

各投稿は「カード」形式で表示されます。

```
┌──────────────────────────────────────┐
│ 🔵 ユーザー名 · 2時間前               │
│                                      │
│ 投稿の本文がここに表示されます。        │
│ 画像や動画もここに。                   │
│                                      │
│ ⭐️×3  💬2  🔁1  📤                   │
└──────────────────────────────────────┘
```

## スクロール

### 下にスクロール

過去の投稿を読み込みます（無限スクロール）。

### 上に戻る

新しい投稿を確認できます。見落とした投稿がある場合は「ギャップ」として検出され、クリックで読み込めます。

## 投稿をクリック

投稿カードをクリックすると詳細ページが開き、返信なども見られます。

## フィルター

画面上部のフィルターで表示を絞り込めます。

→ [フィルター](../features/filters.md)

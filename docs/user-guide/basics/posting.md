# 投稿

テキストや画像を投稿する方法です。

## 基本的な投稿

1. 画面上部のテキストエリアに入力
2. `#mypace` タグは自動で付きます
3. 「投稿」ボタンをクリック

## 画像を添付

1. 📷ボタンをクリック
2. 画像を選択
   - クリックまたはドラッグ＆ドロップでアップロード
   - 📋ボタンでクリップボードから貼り付け
   - URLを直接入力
3. トリミングして「Add」

## 長文モード

長い記事を書くときは「LONG」ボタンで長文エディタに切り替え。

- CodeMirrorエディタ
- Vimモード対応（設定から有効化）
- プレビュー機能

## 投稿の編集

1. 自分の投稿のメニュー（⋯）をクリック
2. 「編集」を選択
3. 内容を修正して「更新」

> **注意**: Nostrの性質上、編集は「新しい投稿で置き換え」として処理されます。

## 投稿の削除

1. 自分の投稿のメニュー（⋯）をクリック
2. 「削除」を選択
3. 確認して「削除」

> **注意**: Nostrの性質上、他のリレーに配信された投稿が完全に消える保証はありません。

## 下書き

入力中の内容は自動保存されます。ページを閉じても、次回アクセス時に復元されます。

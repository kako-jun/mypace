# ステッカー

投稿カードの上に画像を自由配置できる「シール」機能です。

## 特徴

- 投稿に視覚的なアクセントを追加
- 自由な位置・サイズ・回転で配置
- 最大10枚まで

## 使い方

1. 投稿フォームのステッカーボタンをクリック
2. 画像を選択（アップロード、URL入力、または履歴から）
3. 「Sticker」ボタンをクリック
4. プレビューでシールをドラッグして配置

### 画像の選択方法

| 方法 | 説明 |
|------|------|
| Upload | ファイルをアップロード |
| URL | 既存の画像URLを入力 |
| 履歴 | 過去に使った画像から選択 |

### シールの編集

シールをクリックすると編集モードになります。

| 操作 | 方法 |
|------|------|
| 移動 | ドラッグ |
| サイズ変更 | 四隅の青い丸をドラッグ |
| 回転 | 上部の緑の丸をドラッグ |
| 前面/背面 | 下部のトグルスイッチ |
| 削除 | 右上の×ボタン |

### レイヤー

- **前面（Front）**: テキストの上に表示
- **背面（Back）**: テキストの後ろに表示

## お絵かきからシールに

[お絵かき機能](./drawing.md)で描いた絵をシールとして使えます。

1. お絵かきを完成させる
2. 「シール」ボタンをクリック

## 画像の一部を隠す

写真の都合の悪い部分をステッカーで隠したい場合は、[画像編集](./image-crop.md)を使います。アップロード前にステッカーを画像に焼き込むことができます。

## 他のNostrクライアントでの表示

シールはMY PACE独自機能のため、他のクライアントでは表示されません。本文のみが見えます。

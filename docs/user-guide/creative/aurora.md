# オーロラ

投稿カードの背景に4隅グラデーションを設定する機能です。

## 特徴

- PS1 FF7のウィンドウカラー風
- 4隅それぞれに色を指定
- テキスト色は自動調整

## 使い方

投稿時にオーロラ設定を開き、4隅の色を選びます。

```
┌──────────────┬──────────────┐
│   左上の色    │   右上の色    │
│              │              │
├──────────────┼──────────────┤
│   左下の色    │   右下の色    │
│              │              │
└──────────────┴──────────────┘
```

## テキスト色の自動調整

背景が暗い色の組み合わせなら白文字、明るい色の組み合わせなら黒文字に自動で切り替わります。

## 他のNostrクライアントでの表示

オーロラはMY PACE独自機能のため、他のクライアントでは通常のテキスト投稿として表示されます。

# お絵かき

Splatoon風の制限付きお絵かき機能です。

## 特徴

- **42秒の時間制限**: 緊張感のあるドローイング
- **シンプルなツール**: 3色 × 3サイズ
- **味のある絵**: 制約があるから個性が出る

## 仕様

| 項目 | 値 |
|------|-----|
| キャンバスサイズ | 320×120ピクセル |
| 色 | 黒 / グレー / 白（消しゴム） |
| ペンの太さ | 小 / 中 / 大 |
| 制限時間 | 42秒 |

## 使い方

1. 投稿フォームの🖌️（お絵かき）ボタンをクリック
2. キャンバスに描く（最初のストロークでタイマー開始）
3. 完成したら：
   - **埋め込み**: 画像として投稿に追加
   - **シール**: ステッカーとして配置

### ツールバー

- **色選択**: 黒/グレー/白をクリック（白は消しゴム代わり）
- **太さ選択**: 小/中/大をクリック
- **塗りつぶし**: キャンバス全体を選択色で塗りつぶし
- **スクリーントーン**: 点々パターンで塗りつぶし（漫画風）
- **Undo**: 直前のストロークを取り消し

### タイマー

- モーダルを開いただけでは開始しません
- 最初に描き始めたときにスタート
- 残り10秒で赤く点滅
- 0秒になっても描きかけは完成可能

## 他のNostrクライアントでの表示

お絵かきは通常の画像として投稿されるので、どのクライアントでも表示されます。

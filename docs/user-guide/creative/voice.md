# ボイスメモ

短い音声メッセージを投稿に添付する機能です。

## 特徴

- **押し続けて録音**: LINE/WhatsApp風の操作
- **最大10秒**: 短くて気軽
- **テキストを打つのが面倒な時に**

## 使い方

1. 投稿フォームの🎤（録音）ボタンをクリック
2. マイクボタンを**押し続けて**録音
3. 離すと録音停止
4. ▶で再生して確認
5. 「Add to Post」で投稿に追加

### 操作

| 操作 | 説明 |
|------|------|
| 押し続ける | 録音開始 |
| 離す | 録音停止 |
| ▶ | 録音内容を再生 |
| Re-record | 最初からやり直し |
| Add to Post | 投稿に追加 |

### タイマー

- 録音開始と同時にスタート
- 残り3秒で赤く点滅
- 10秒で自動停止

## マイクへのアクセス許可

初回は「マイクへのアクセスを許可しますか？」というダイアログが表示されます。「許可」をクリックしてください。

## 他のNostrクライアントでの表示

音声はURLとして投稿されるので、音声プレーヤーに対応したクライアントで再生できます。

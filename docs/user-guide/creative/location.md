# 位置情報

投稿に場所情報を添付する機能です。

## こんなときに

- 「ここに行った」感想を書きたい
- お店のレビューを投稿したい
- 旅行先の記録を残したい

## 使い方

1. 投稿フォームの📍（座標）ボタンをクリック
2. 地名や施設名で検索
3. 候補から選択
4. 必要に応じて地図上でピンをドラッグして微調整
5. 「Add Location」をクリック

### 精度の選択

| 設定 | 精度 | 用途 |
|------|------|------|
| Area | ±610m | 町・エリア |
| Street | ±76m | 通り |
| Building | ±19m | 建物（デフォルト） |
| Precise | ±2m | 正確な位置 |

## 表示

投稿に場所が追加されると、カードの下部に📍アイコンと地名が表示されます。クリックで地図を開きます。

```
┌──────────────────────────────────────┐
│ @username · 2時間前                   │
│                                      │
│ 金閣寺に行ってきた！                   │
│                                      │
│ 📍 金閣寺  ← クリックで地図を開く      │
└──────────────────────────────────────┘
```

## 他のNostrクライアントでの表示

位置情報タグはNostr標準（NIP-52）に準拠しているため、対応クライアントでは地図表示される可能性があります。

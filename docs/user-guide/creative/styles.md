# スタイル（フォント・配置）

投稿のテキストに色・サイズ・配置を指定する機能です。

---

## アライメント構文

行頭にマーカーを置くことで、配置を指定できます。

| 記法 | 意味 |
|------|------|
| `<<` | 左寄せ |
| `>>` | 右寄せ |
| `><` | センタリング |
| `<>` | 左右分割 |

### 使用例

```
<< 左寄せテキスト
>> 右寄せテキスト
>< センタリング
```

### 左右分割

`|` で区切って左右に配置:

```
<> りんご | ¥100
<> みかん | ¥80
```

結果:
```
┌────────────────────────┐
│ りんご           ¥100  │
│ みかん            ¥80  │
└────────────────────────┘
```

### 会話風（LINE風）

```
<< こんにちは
>> やあ！
<< 元気？
>> うん！
```

> **注意**: マーカーと内容の間にはスペースが必要です。`<<テキスト` ではなく `<< テキスト` と書きます。

## Font構文

HTMLの`<font>`タグ風の記法で、文字色とサイズを指定できます。

### 色を指定

```
<font color="red">赤い文字</font>
<font color="#FF5500">カスタム色</font>
```

**使える色名**: red, blue, green, yellow, orange, purple, pink, cyan, navy, teal, white, black, gray など

### サイズを指定

```
<font size="1">極小</font>
<font size="2">小さい</font>
<font size="3">普通</font>
<font size="4">大きい</font>
<font size="5">特大</font>
```

### 色とサイズを同時に

```
<font color="red" size="5">赤くて大きい</font>
```

### 使用例：見出し風

```
>< <font color="navy" size="5">第一章 はじまり</font>

ここから本文が始まる。
途中で<font color="red">重要な部分</font>を強調する。
```

### 使用例：会話の書き分け

```
<font color="red">太郎:</font>「おはよう」
<font color="blue">花子:</font>「おはよう。今日は早いね」
```

## 他のNostrクライアントでの表示

- **アライメント**: マーカー（`<<`, `>>` 等）がそのまま表示されます
- **Font構文**: タグが除去されるか、そのまま表示されます

いずれにせよ、内容は読めます。

# メディア編集

アップロード前に画像をトリミングしたり、動画をアニメーション画像に変換できます。

---

## 画像編集

画像をトリミングしたり、ステッカーで隠したい部分をカバーできます。

### 特徴

- 不要な部分をカット（トリミング）
- ステッカーで隠したい部分をカバー
- 1画面でクロップとステッカー配置を同時操作
- アップロード前に編集できる

### 使い方

1. 画像をアップロード
2. 自動で画像編集モーダルが開く
3. クロップ範囲を指定 / ステッカーを配置
4. 「Add」で編集を確定してアップロード

#### クロップ操作

| 操作 | 説明 |
|------|------|
| ドラッグ | クロップ範囲を選択 |
| 角をドラッグ | サイズ変更 |
| 範囲内をドラッグ | 位置移動 |

#### ステッカーで隠す

モーダル下部にステッカー履歴が表示されます。

1. 履歴からステッカーをタップして追加
2. ステッカーをドラッグして隠したい場所に配置
3. 四隅のハンドルでサイズ変更
4. 上部のハンドルで回転
5. 左上の×で削除

> ステッカーは画像に焼き込まれるため、アップロード後は編集できません。

### 対象

- 通常の画像添付
- ステッカー用画像

### 注意

- 編集はアップロード前のみ
- アニメーション画像（GIF/APNG/WebP）は編集をスキップして直接アップロードされます

---

## 動画→アニメ画像変換

動画ファイルをアニメーションWebPに変換できます。サーバーを使わず、すべてブラウザ内で処理されます。

### 特徴

- 動画の一部（最大10秒）を切り出してアニメ画像に変換
- クロップ（範囲選択）で不要部分をカット
- 自動リサイズ（最大320px）で軽量化
- WebP形式で高圧縮・高画質
- 変換後は自動ループ再生

### 使い方

1. 動画ファイルをアップロード（ドラッグ&ドロップまたはファイル選択）
2. 動画編集モーダルが開く
3. **時間範囲を選択**
   - 緑のスライダー：開始点
   - 赤のスライダー：終了点
   - 再生ボタンで選択範囲をプレビュー
4. **（オプション）クロップ範囲を選択**
   - 動画上でドラッグして範囲を描画
   - 「Clear」で解除
5. 「Convert」をクリック
6. 変換完了後、自動でアップロード

### 制限

| 項目 | 値 |
|------|-----|
| 最大秒数 | 10秒 |
| 出力解像度 | 最大320px（自動縮小） |
| フレームレート | 10fps |
| 出力形式 | WebP（アニメーション） |

### 対応形式

ブラウザが再生できる動画形式すべて：

- MP4 (H.264)
- WebM (VP8/VP9)
- MOV
- など

### 注意

- 長い動画でも、指定した10秒分のみ処理されるのでメモリ効率は良好
- 変換処理中は進捗バーが表示されます
- クリップボードからの動画ペーストには非対応（ブラウザ制限）

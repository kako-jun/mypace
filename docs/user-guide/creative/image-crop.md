# 画像トリミング

アップロード前に画像をトリミングする機能です。

## 特徴

- 不要な部分をカット
- アップロード前に編集できる
- スマホで撮った写真の余白を削除

## 使い方

1. 画像をアップロード
2. 自動でクロップモーダルが開く
3. ドラッグで選択範囲を指定
4. 「Add」でトリミング＆アップロード

### 操作

| 操作 | 説明 |
|------|------|
| ドラッグ | クロップ範囲を選択 |
| 角をドラッグ | サイズ変更 |
| 範囲内をドラッグ | 位置移動 |
| Cancel | クロップをキャンセル |
| Add | 切り取ってアップロード |

## 対象

- 通常の画像添付
- ステッカー用画像

> **注意**: トリミングはアップロード前のみ。アップロード後は変更できません。

# 画像編集

アップロード前に画像をトリミングしたり、ステッカーで隠したい部分をカバーできます。

## 特徴

- 不要な部分をカット（トリミング）
- ステッカーで隠したい部分をカバー
- 1画面でクロップとステッカー配置を同時操作
- アップロード前に編集できる

## 使い方

1. 画像をアップロード
2. 自動で画像編集モーダルが開く
3. クロップ範囲を指定 / ステッカーを配置
4. 「Add」で編集を確定してアップロード

### クロップ操作

| 操作 | 説明 |
|------|------|
| ドラッグ | クロップ範囲を選択 |
| 角をドラッグ | サイズ変更 |
| 範囲内をドラッグ | 位置移動 |

### ステッカーで隠す

モーダル下部にステッカー履歴が表示されます。

1. 履歴からステッカーをタップして追加
2. ステッカーをドラッグして隠したい場所に配置
3. 四隅のハンドルでサイズ変更
4. 上部のハンドルで回転
5. 左上の×で削除

> ステッカーは画像に焼き込まれるため、アップロード後は編集できません。

## 対象

- 通常の画像添付
- ステッカー用画像

## 注意

- 編集はアップロード前のみ
- アニメーション画像（GIF/APNG/WebP）は編集をスキップして直接アップロードされます

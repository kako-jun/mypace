## 使い方

### [3. 便利な機能](./features/)

日々の利用を便利にする機能。

- [フィルター](./features/filters.md) - 表示を絞り込む
- [ミュート](./features/mute.md) - 特定ユーザーを非表示
- [共有](./features/share.md) - URL・Markdownで共有
- [埋め込み](./features/embed.md) - YouTube、Twitter等の表示
- [ファイル読込](./features/file-import.md) - ローカルファイルから投稿
- [アップロード履歴](./features/upload-history.md) - 画像の管理

### [4. クリエイティブ機能](./creative/)

表現の幅を広げる機能。

- [お絵かき](./creative/drawing.md) - Splatoon風42秒ドローイング
- [ボイスメモ](./creative/voice.md) - 10秒の音声録音
- [ステッカー](./creative/sticker.md) - 投稿にシールを貼る
- [オーロラ](./creative/aurora.md) - カード背景のグラデーション
- [スタイル](./creative/styles.md) - フォント・配置のカスタマイズ
- [画像トリミング](./creative/image-crop.md) - アップロード前の編集
- [位置情報](./creative/location.md) - 座標を投稿に追加

### [5. カスタマイズ](./customize/)

MY PACEを自分好みに。

- [プロフィール](./customize/profile.md) - 名前・アバター・自己紹介
- [設定エクスポート](./customize/settings-export.md) - 設定の保存・復元

### [6. 上級者向け](./advanced/)

パワーユーザーのための機能。

- [鍵の管理](./advanced/keys.md) - エクスポート・インポート
- [NIP-07拡張](./advanced/nip-07.md) - ブラウザ拡張で安全に
- [スーパーメンション](./advanced/super-mention.md) - 万物への言及

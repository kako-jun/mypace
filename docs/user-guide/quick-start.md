# クイックスタート

5分でMY PACEを使い始めましょう。

## 1. サイトにアクセス

[https://mypace.llll-ll.com](https://mypace.llll-ll.com) にアクセスします。

## 2. アカウント作成（自動）

初回アクセス時に自動でアカウントが作成されます。登録やログインは不要です。

> **仕組み**: 秘密鍵がブラウザに自動生成され、安全に保存されます。
> この鍵があなたのアイデンティティです。

## 3. プロフィールを設定

1. 右上のアバターアイコンをクリック
2. プロフィールページで「編集」をクリック
3. 名前を入力（必須）
4. 「保存」をクリック

> **ヒント**: アバター画像や自己紹介は後からでも設定できます。

## 4. 最初の投稿

1. 画面上部の入力欄にテキストを入力
2. `#mypace` タグは自動で付きます
3. 「投稿」ボタンをクリック

おめでとうございます！最初の投稿が完了しました。

## 5. タイムラインを見る

- タイムラインには他のユーザーの投稿も表示されます
- 星（ステラ）をクリックしてリアクションを送れます
- 投稿をクリックすると詳細が見られます

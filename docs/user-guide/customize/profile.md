# プロフィール

自分のプロフィールを設定する方法です。

## アクセス方法

1. 右上のアバターアイコンをクリック
2. プロフィールページで「編集」をクリック

## 設定項目

| 項目 | 必須 | 説明 |
|------|------|------|
| 名前 | ○ | 表示名 |
| アバター | - | プロフィール画像 |
| バナー | - | ヘッダー画像 |
| 自己紹介 | - | 自由記述 |
| Webサイト | - | リンクURL |
| NIP-05 | - | 認証用アドレス |
| Lightning Address | - | 投げ銭受け取り用 |

## NIP-05認証

NIP-05アドレス（例: `username@example.com`）を設定すると、プロフィールに✓マークが表示されます。

### 設定方法

1. NIP-05サービスでアドレスを取得
2. プロフィール編集画面で「NIP-05」に入力
3. 保存

## ユーザー通し番号

初めて`#mypace`タグ付き投稿をすると、参加順の通し番号が付与されます。プロフィールに「Thanks #N」として表示されます。

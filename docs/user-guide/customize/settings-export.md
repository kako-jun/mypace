# 設定エクスポート

複数デバイス間で設定を同期するための機能です。

## エクスポートできる設定

| 設定 | 説明 |
|------|------|
| テーマ | ダーク/ライトモード |
| グラデーション色 | 4隅の色設定 |
| フィルタ設定 | NGワード、NGタグ、言語など |
| フィルタプリセット | 保存済みのプリセット |
| ミュートリスト | ミュートしたユーザー |

## エクスポートされない設定

| 設定 | 理由 |
|------|------|
| 秘密鍵 | セキュリティのため |
| 下書き | デバイス固有のため |

> **重要**: 秘密鍵は別途バックアップしてください。→ [鍵の管理](../advanced/keys.md)

## 使い方

### エクスポート

1. 設定画面を開く
2. 「Export」ボタンをクリック
3. `mypace-settings.json` がダウンロードされる

### インポート

1. 設定画面を開く
2. 「Import」ボタンをクリック
3. JSONファイルを選択
4. 設定が復元される

## 活用例

### PC → スマホ

1. PCで「Export」
2. クラウドストレージやメッセージアプリでスマホに送信
3. スマホで「Import」

### バックアップ

定期的にエクスポートして安全な場所に保管しましょう。

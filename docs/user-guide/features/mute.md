# ミュート

特定ユーザーの投稿を非表示にする機能です。

## ミュートに追加する

1. ヘッダーのフィルタボタンをクリック
2. 「Mute List」ボタンをクリック
3. npubアドレスを入力してEnterまたは「Add」

> **npubアドレス**: ユーザーのプロフィールページなどで確認できる `npub1...` で始まるアドレスです。

## ミュートを解除する

1. 「Mute List」ポップアップを開く
2. 解除したいユーザーの × ボタンをクリック

## プライバシー

- ミュートリストはあなたのブラウザに保存されます
- 共有URLにミュート情報は含まれません
- ミュートしていることは他のユーザーにはわかりません

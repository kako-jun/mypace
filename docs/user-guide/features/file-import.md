# ファイル読み込み

ローカルのMarkdown/テキストファイルをエディタに読み込む機能です。

## 使い方

### クリックで読み込み

1. エディタの📁（ファイル）ボタンをクリック
2. `.md` または `.txt` ファイルを選択
3. 内容がエディタに読み込まれる

### ドラッグ＆ドロップ

1. ファイルを📁ボタンにドラッグ
2. ボタンの枠線が変化したらドロップ
3. 内容がエディタに読み込まれる

> **注意**: ファイルを読み込んでも自動投稿されません。内容を確認・編集してから投稿してください。

## 対応形式

| 拡張子 | 説明 |
|--------|------|
| `.md` | Markdownファイル |
| `.txt` | テキストファイル |
| `.markdown` | Markdownファイル |

## 活用例

- 事前に書いた記事を投稿
- 他のエディタで作成した下書きをインポート
- Obsidian、Notionからのエクスポートを投稿

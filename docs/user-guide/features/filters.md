# フィルター

タイムラインに表示する投稿を絞り込む機能です。

## 投稿種別フィルタ（SNS/Blog）

表示する投稿の種別を選べます。

| 設定 | 表示内容 |
|------|----------|
| SNS オン | 短文投稿（Twitterのような投稿） |
| Blog オン | 長文記事（ブログ） |

**使い方**
1. フィルタパネルを開く
2. 回路図UIでSNS/Blogのスイッチを切り替え
3. 「Save」をクリック

> **ヒント**: ブログだけを読みたいときは、SNSをオフにしてBlogだけオンにしましょう。

## MY PACE Only フィルタ

- オン: `#mypace` タグ付きの投稿のみ表示（デフォルト）
- オフ: すべての投稿を表示

MY PACEコミュニティの投稿だけを見たいときはオンのままに。

## テキストフィルタ（OK/NG）

キーワードやタグで投稿を絞り込めます。

### OKワード（検索）

指定したワードを**すべて含む**投稿のみ表示。

```
例: 猫 犬
→ 「猫と犬が好き」は表示、「猫が好き」だけでは非表示
```

### OKタグ

指定したタグを**すべて含む**投稿のみ表示。

```
例: mypace nostr
→ #mypace と #nostr の両方があるもののみ
```

### NGワード

指定したワードを**いずれか含む**投稿を非表示。

```
例: 広告 PR
→ 「広告です」「PR案件」を含む投稿は非表示
```

### NGタグ

指定したタグを**いずれか含む**投稿を非表示。

```
例: nsfw ad
→ #nsfw または #ad を含む投稿は非表示
```

**入力方法**: スペース区切りで複数指定できます。

## スマートフィルタ

サーバー側で自動的に不要なコンテンツを除外します。

| 設定 | デフォルト | 効果 |
|------|-----------|------|
| Hide Ads | オン | 広告・スパム・仮想通貨関連を非表示 |
| Hide NSFW | オン | 成人向けコンテンツを非表示 |

**使い方**
1. フィルタパネルの「Smart Filter」をクリック
2. チェックボックスで切り替え

> **注意**: これらをオフにすると、広告やNSFWコンテンツが表示されます。

### 言語フィルタ

投稿の言語で絞り込めます。初回は自動的にブラウザの言語が設定されます。

---

## プリセット

よく使うフィルタ設定を保存して、すぐに呼び出せます。

### 保存する

1. フィルタを設定
2. 💾（保存）ボタンをクリック
3. 名前を入力して保存

### 呼び出す

1. ドロップダウンからプリセットを選択
2. フィルタが自動で適用される

### 削除する

1. 削除したいプリセットを選択
2. 🗑️（削除）ボタンを2回クリック（確認のため）

> **ヒント**: 最大10個まで保存できます。

---

## フィルタの保存先

| フィルタ | 保存先 | 共有 |
|---------|--------|------|
| OKワード・タグ | URL | URLをコピーすれば共有可能 |
| NGワード・タグ | ブラウザ | 個人設定、共有されない |
| スマートフィルタ | ブラウザ | 個人設定、共有されない |
| プリセット | ブラウザ | 個人設定、共有されない |

# アップロード履歴

nostr.buildにアップロードしたファイルの履歴を管理する機能です。

## 特徴

- アップロードしたファイルの一覧を確認
- ファイルをnostr.buildから削除
- デバイスが変わっても履歴を維持

## アクセス方法

- 設定 → Account → Upload History →
- 画像選択モーダルの🕐アイコン
- お絵かきモーダルの🕐アイコン
- ボイスメモモーダルの🕐アイコン

## 操作

| 操作 | 機能 |
|--------|------|
| サムネイルクリック | 画像/動画/音声をプレビュー再生 |
| Copy | URLをクリップボードにコピー |
| DELETE | nostr.buildからファイルを削除 |
| × ボタン（画像選択モーダル内） | 履歴から削除（nostr.buildからは削除されない） |

## ファイルの削除

「DELETE」ボタンでnostr.buildからファイルを削除できます。画像選択モーダル内の履歴に表示される「×」ボタンは履歴から削除するのみで、サーバーからは削除されません。

> **注意**: 削除後もCDNキャッシュにより、数分〜数時間は画像が表示される場合があります。実際のファイルは削除済みで、キャッシュが切れれば消えます。

## 対象ファイル

以下のアップロードが自動的に履歴に保存されます:

- 画像添付
- ステッカー画像
- お絵かき
- ボイスメモ

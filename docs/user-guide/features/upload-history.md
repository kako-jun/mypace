# アップロード履歴

nostr.buildにアップロードしたファイルの履歴を管理する機能です。

## 特徴

- アップロードしたファイルの一覧を確認
- ファイルをnostr.buildから削除
- デバイスが変わっても履歴を維持

## アクセス方法

- 設定 → Account → Upload History →
- 画像選択モーダルの🕐アイコン
- お絵かきモーダルの🕐アイコン
- ボイスメモモーダルの🕐アイコン

## 操作

| ボタン | 機能 |
|--------|------|
| Copy | URLをクリップボードにコピー |
| Delete | nostr.buildからファイルを削除 |
| × | 履歴から削除 |

## ファイルの削除

「Delete」ボタンでnostr.buildからファイルを削除できます。

> **注意**: 削除後もCDNキャッシュにより、数分〜数時間は画像が表示される場合があります。実際のファイルは削除済みで、キャッシュが切れれば消えます。

## 履歴からの削除

「×」ボタンで履歴から削除できますが、nostr.buildからファイルを削除する情報も失われます。

> **警告**: 履歴から削除すると、後からnostr.buildのファイルを削除できなくなる可能性があります。

## 対象ファイル

以下のアップロードが自動的に履歴に保存されます:

- 画像添付
- ステッカー画像
- お絵かき
- ボイスメモ

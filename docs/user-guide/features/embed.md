# 埋め込み

外部サービスのコンテンツをタイムラインに表示する機能です。

## 対応サービス

| サービス | 例 |
|---------|-----|
| YouTube | 動画、Shorts |
| Twitter/X | ツイート |
| Instagram | 投稿 |
| TikTok | 動画 |
| Spotify | トラック、アルバム、プレイリスト |

## 使い方

投稿にURLを含めるだけで、自動的に埋め込み表示されます。

```
この曲おすすめ！
https://open.spotify.com/track/xxx

#mypace
```

## OGPプレビュー

対応サービス以外のURLも、OGP情報（タイトル・画像）がプレビュー表示されます。

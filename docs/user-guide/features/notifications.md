# 通知

自分の投稿への反応を確認できます。

## 通知の種類

| 種類 | アイコン | 説明 |
|------|---------|------|
| ステラ | ★ | 投稿にステラが付いた |
| リプライ | 💬 | 投稿へのリプライが届いた |
| リポスト | 🔁 | 投稿がリポストされた |

## 確認方法

1. ヘッダー右側の **ベルアイコン** をクリック
2. 通知一覧がポップアップ表示される
3. 通知をタップすると該当の投稿に移動

## ベルアイコンの表示

| 状態 | 表示 |
|------|------|
| 未読あり | 塗りつぶし（青） |
| 未読なし | 線のみ（白） |

未読件数は表示しません（追い立てないポリシー）。

## 既読の仕組み

- ポップアップを開いただけでは既読になりません
- **通知をタップして投稿に移動すると**既読になります
- 既読の通知は薄く表示されます

## 通知の集約

同じ投稿へのステラやリポストは1行にまとめて表示されます。

```
例: alice, bob 他3人が投稿Xにステラ
```

リプライは個別に表示されます（内容を見逃さないため）。

## 注意事項

- **MY PACE経由のみ**: 他のNostrクライアントからの反応は通知されません
- **保存件数**: 最新50件まで保存されます

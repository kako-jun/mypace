# 通知

自分の投稿への反応を確認できます。

## 通知の種類

| 種類 | アイコン | 説明 |
|------|---------|------|
| ステラ | ★ | 投稿にステラが付いた |
| リプライ | 💬 | 投稿へのリプライが届いた |
| リポスト | 🔁 | 投稿がリポストされた |

## 確認方法

1. ヘッダー右側の **ベルアイコン** をクリック
2. 通知一覧がポップアップ表示される
3. 通知をタップすると該当の投稿に移動

## ベルアイコンの表示

| 状態 | 表示 |
|------|------|
| 未読あり | 塗りつぶし（青） |
| 未読なし | 線のみ（白） |

未読件数は表示しません（追い立てないポリシー）。

## 既読の仕組み

- ポップアップを開いただけでは既読になりません
- **通知をタップして投稿に移動すると**既読になります
- 既読の通知は薄く表示されます

## 通知の集約

同じ投稿へのステラやリポストは1行にまとめて表示されます。

```
例: 2人から +⭐4 +💙2
```

ステラ通知は「何人から」「色ごとのステラ数」で表示されます。
カラーステラにも対応しています（黄・緑・赤・青・紫）。

リプライは個別に表示されます（内容を見逃さないため）。

## プッシュ通知

MY PACEを閉じていても、ブラウザからポップアップで通知を受け取れます。

### 有効化の手順

1. **SETTINGS** を開く
2. **Push Notifications** のトグルをONにする
3. ブラウザの「通知を許可しますか？」で **許可** をタップ

### 通知オプション

| オプション | 説明 |
|-----------|------|
| All notifications | ステラ・リプライ・リポスト全部通知 |
| Replies only | リプライだけ通知 |

### 通知の動作

- **種類ごとに1つ**: ステラ・リプライ・リポストそれぞれ最大1つの通知（スパム防止）
- **同じ種類は上書き**: 連続でステラが来ても通知は増えず、既存の通知が更新される
- **タップでアプリ起動**: 通知をタップするとMY PACEが開く

### 対応ブラウザ

- Chrome / Edge / Firefox / Safari（iOS 16.4以降）
- ブラウザの通知設定がブロックされていると使用できません

## 注意事項

- **MY PACE経由のみ**: 他のNostrクライアントからの反応は通知されません
- **保存件数**: 最新50件まで保存されます

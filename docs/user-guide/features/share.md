# 共有

投稿をSNS・URL・Markdownで共有する機能です。

## 共有メニュー

投稿の共有ボタンをクリックすると、以下のオプションが表示されます。

### Share to SNS

> **注意**: この機能は自分の投稿でのみ表示されます（なりすまし防止のため）。

外部SNSに投稿を共有します。MY PACEで書いた投稿を他のSNSにも展開したい場合に便利です。

| SNS | 説明 |
|-----|------|
| X | X（旧Twitter）の投稿画面が開きます |
| Bluesky | Blueskyの投稿画面が開きます |
| Threads | Threadsの投稿画面が開きます |

> **ヒント**: MY PACEをメインに使いつつ、反響を得たい投稿だけを他SNSに展開する使い方がおすすめです。

### 自動変換

SNS共有時に、MY PACE独自の機能は自動的に変換されます。

| MY PACE機能 | SNSでの表示 |
|-------------|-------------|
| スーパーメンション `@@対象` | ハッシュタグ `#対象` |
| 派生パス `@@作品/キャラ` | `#作品_キャラ` |
| Nostr ハッシュタグ（tタグ） | ハッシュタグ `#タグ名` |
| 位置情報 | `📍 地名` + OpenStreetMap URL |
| 画像・音声 | URLとして表示 |

- 投稿の末尾には、MY PACEのオリジナル投稿へのリンクが自動で追加されます
- `#mypace` タグが自動で付与されます（MY PACEからの投稿であることを示す）
- 他のNostrクライアントで付けたハッシュタグも保持されます

### 長文の分割投稿

文字数制限を超える長文は、複数パートに分割できます。

```
Share to SNS → X
  ├── Full（全文）
  ├── 1/3
  ├── 2/3
  └── 3/3
```

- 段落（空行）、改行、句読点を区切りとして自動分割
- パート番号 `(1/3)` が自動で付与
- MY PACE URLは最後のパートに挿入
- X は URL を 23文字として計算（t.co 短縮を考慮）
- Bluesky は URL を 22文字として計算

> **ヒント**: 分割投稿を使えば、MY PACEで書いた長文エッセイをXにスレッド形式で展開できます。

### Share URL

投稿のURLを共有します。

| オプション | 説明 |
|-----------|------|
| Copy URL | MY PACE の URL をクリップボードにコピー |
| Copy Nostr ID | Nostr ID（`note1...`形式）をコピー |
| Share to Apps | 端末の共有ダイアログを開く（モバイル向け） |

**Copy Nostr ID** でコピーした ID は、他の Nostr クライアント（Damus, Amethyst, Snort など）で直接開くことができます。

> **注意**: Share to Apps は Web Share API 対応環境でのみ動作します。非対応の場合は自動でクリップボードにコピーされます。

### Share Content

投稿の内容（テキスト）を共有します。

| オプション | 説明 |
|-----------|------|
| Copy Markdown | テキストをクリップボードにコピー |
| Open Markdown URL | 生テキストをブラウザで開く |
| Download Markdown | `.md` ファイルとしてダウンロード |

> **ヒント**: 長文投稿も完全なテキストで共有できます。

## Nostr クライアント間の相互運用

MY PACE は他の Nostr クライアントとの相互運用をサポートしています。

### MY PACE → 他クライアント

**Share URL → Copy Nostr ID** で `note1...` 形式の ID をコピーし、他の Nostr クライアントに貼り付けて開けます。

### 他クライアント → MY PACE

`nostr:note1...` や `nostr:nevent1...` 形式のリンクをクリックすると、MY PACE で開くことができます（PWA インストール時）。

対応形式:
- `nostr:note1...` - 投稿 ID
- `nostr:nevent1...` - 投稿 ID + リレーヒント
- `nostr:npub1...` - ユーザー ID
- `nostr:nprofile1...` - ユーザー ID + リレーヒント

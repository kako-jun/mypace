# 共有

投稿をSNS・URL・Markdownで共有する機能です。

## 共有メニュー

投稿の共有ボタンをクリックすると、以下のオプションが表示されます。

### Share to SNS

外部SNSに投稿を共有します。MY PACEで書いた投稿を他のSNSにも展開したい場合に便利です。

| SNS | 説明 |
|-----|------|
| X | X（旧Twitter）の投稿画面が開きます |
| Bluesky | Blueskyの投稿画面が開きます |
| Threads | Threadsの投稿画面が開きます |

> **ヒント**: MY PACEをメインに使いつつ、反響を得たい投稿だけを他SNSに展開する使い方がおすすめです。

### 自動変換

SNS共有時に、MY PACE独自の機能は自動的に変換されます。

| MY PACE機能 | SNSでの表示 |
|-------------|-------------|
| スーパーメンション `@@対象` | ハッシュタグ `#対象` |
| 派生パス `@@作品/キャラ` | `#作品_キャラ` |
| 位置情報 | `📍 地名` + OpenStreetMap URL |
| 画像・音声 | URLとして表示 |

投稿の末尾には、MY PACEのオリジナル投稿へのリンクが自動で追加されます。

### 長文の分割投稿

文字数制限を超える長文は、複数パートに分割できます。

```
Share to SNS → X
  ├── Full（全文）
  ├── 1/3
  ├── 2/3
  └── 3/3
```

- 段落（空行）、改行、句読点を区切りとして自動分割
- パート番号 `(1/3)` が自動で付与
- MY PACE URLは最後のパートに挿入
- X は URL を 23文字として計算（t.co 短縮を考慮）

> **ヒント**: 分割投稿を使えば、MY PACEで書いた長文エッセイをXにスレッド形式で展開できます。

### Share URL

投稿のURLを共有します。

- **モバイル**: 端末の共有ダイアログが開きます
- **デスクトップ**: URLがクリップボードにコピーされます

### Share Content

投稿の内容（テキスト）を共有します。

| オプション | 説明 |
|-----------|------|
| Copy Markdown | テキストをクリップボードにコピー |
| Open Markdown URL | 生テキストをブラウザで開く |
| Download Markdown | `.md` ファイルとしてダウンロード |

> **ヒント**: 長文投稿も完全なテキストで共有できます。

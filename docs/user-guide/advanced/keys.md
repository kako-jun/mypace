# 鍵の管理

秘密鍵のエクスポートとインポートについて。

## 秘密鍵とは

MY PACEでは、従来のユーザー名・パスワードの代わりに「秘密鍵」があなたのアイデンティティです。

- 初回アクセス時に自動生成
- ブラウザに保存
- **この鍵を失うとアカウントにアクセスできなくなります**

## 認証方式

MY PACEでは2つの認証方式を選べます。

### 1. 秘密鍵（nsec）方式（デフォルト）

秘密鍵をブラウザのローカルストレージに暗号化して保存する方式です（AES-GCM）。複数の秘密鍵を登録し、切り替えて使うこともできます。

- **メリット**: シンプル、追加のソフトウェア不要、スマートフォンでも使える
- **デメリット**: 暗号化はデバイス固定の難読化であり、完全な保護ではない

### 2. NIP-07拡張方式

秘密鍵をブラウザ拡張機能で管理する方式です。

- **メリット**: より安全（秘密鍵がウェブサイトに渡らない）、複数クライアントで共有可能
- **デメリット**: PCのブラウザでのみ利用可能、拡張機能のインストールが必要

詳しくは [NIP-07拡張](./nip-07.md) を参照してください。

## エクスポート（バックアップ）

秘密鍵を安全な場所にバックアップしましょう。

1. 設定画面を開く
2. 「Your Keys」セクション
3. nsecの「Show」をクリックして表示
4. 「Copy」で `nsec1...` で始まる文字列をコピー
5. **安全な場所に保管**

> **警告**: 秘密鍵は絶対に他人に見せないでください。鍵を持っている人はあなたのアカウントを完全にコントロールできます。

> **注意**: NIP-07拡張を使用している場合、秘密鍵はブラウザに保存されていないため、MY PACEからはエクスポートできません。拡張機能から直接エクスポートしてください。

## インポート・切り替え

複数の秘密鍵を登録し、切り替えて使うことができます。

### 新しい鍵を追加

1. 設定画面を開く
2. 「Your Keys」セクション
3. `nsec1...` 形式の秘密鍵を入力欄に貼り付け
4. 「Add」をクリック（またはEnterキー）

### 鍵を切り替え

複数の鍵が登録されている場合、nsec表示部のドロップダウンから別の鍵を選択できます。ドロップダウンにはnpubの短縮表示に加え、プロフィール名が設定されていればその名前も表示されます。テーマやフィルタなどの設定は鍵間で共有されます。

## 保管場所のおすすめ

- パスワードマネージャー（1Password、Bitwarden等）
- 暗号化されたメモアプリ
- 紙に書いて金庫に保管

> **NG**: メールに送る、クラウドストレージに平文で保存、スクリーンショットを撮る

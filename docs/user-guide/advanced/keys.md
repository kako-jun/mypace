# 鍵の管理

秘密鍵のエクスポートとインポートについて。

## 秘密鍵とは

MY PACEでは、従来のユーザー名・パスワードの代わりに「秘密鍵」があなたのアイデンティティです。

- 初回アクセス時に自動生成
- ブラウザに保存
- **この鍵を失うとアカウントにアクセスできなくなります**

## エクスポート（バックアップ）

秘密鍵を安全な場所にバックアップしましょう。

1. 設定画面を開く
2. 「Account」セクション
3. 「Export Key」をクリック
4. `nsec1...` で始まる文字列をコピー
5. **安全な場所に保管**

> **警告**: 秘密鍵は絶対に他人に見せないでください。鍵を持っている人はあなたのアカウントを完全にコントロールできます。

## インポート

別のデバイスで同じアカウントを使う場合:

1. 設定画面を開く
2. 「Account」セクション
3. 「Import Key」をクリック
4. `nsec1...` 形式の秘密鍵を入力
5. 「Import」をクリック

## 保管場所のおすすめ

- パスワードマネージャー（1Password、Bitwarden等）
- 暗号化されたメモアプリ
- 紙に書いて金庫に保管

> **NG**: メールに送る、クラウドストレージに平文で保存、スクリーンショットを撮る

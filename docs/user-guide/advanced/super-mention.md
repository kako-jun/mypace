# スーパーメンション

万物に対してコメントできる機能です。

## 概要

通常のメンション `@user` は人への呼びかけ。
スーパーメンション `@@` は**万物への呼びかけ**です。

```
@@ハンチョウ

20巻の宮本さん、最高だった
```

## 構文

```
@@対象
```

- `@@` で始まる（ダブルアットマーク）
- 日本語OK
- スペースは自動でアンダースコアに変換

## 使い方

### 方法1: ボタンから

1. エディタ上部の `@@` ボタンをクリック
2. 検索ボックスに入力
3. 候補から選択

### 方法2: 直接入力

1. エディタで `@@` と入力
2. 対象を続けて入力
3. サジェストから選択（または新規作成）

## サジェスト候補

| アイコン | 意味 | Q番号 |
|----------|------|-------|
| ✓ | 確定済みの履歴（Wikidataと紐付け済み） | つく |
| 📌 | 未確定の履歴（Wikidataと紐付けなし） | つかない |
| 🔍 | Wikidata検索結果 | つく |
| ✏️ | 新規作成（独自のメンション） | つかない |

## Wikidataとの連携

候補に表示される `Q番号`（例: `Q12345678`）は Wikidata のIDです。クリックでWikipediaページを開けます。

同じ対象の表記ゆれ（「ハンチョウ」と「1日外出録ハンチョウ」など）も、Q番号で同一視できます。

### Q番号がつくケース

- **Wikidata検索結果（🔍）を選択**: Wikidataと紐付けられ、Q番号がつく
- **確定済み履歴（✓）を選択**: 過去に紐付けたQ番号を継承

### Q番号がつかないケース

- **新規作成（✏️）を選択**: 独自のメンションとしてQ番号なしで作成
- **未確定履歴（📌）を選択**: Wikidataと紐付けられていない履歴
- **URLを入力**: URLへの言及にはQ番号はつかない

### 同じ名前がWikidataにある場合

例えば「Apple」と入力すると、WikidataにはApple Inc.（Q312）が見つかります。

- **Apple Inc.について言及したい** → 検索結果の「Apple」（🔍）を選択 → Q番号つく
- **果物のりんごや、別の意味で言及したい** → 「新規作成」（✏️）を選択 → Q番号つかない

「新規作成」を選べば、Wikidataとは無関係の独自メンションとして扱われます。

## 派生パス（スラッシュ記法）

Wikidataにない下部階層を追加できます。

```
@@ハンチョウ/20巻
@@ハンチョウ/宮本さん
```

### 動作

1. ベース部分（「ハンチョウ」）をWikidata候補から選択
2. スラッシュ `/` を追加して階層を入力
3. **ベースのQ番号を継承**して保存される

これにより、「ハンチョウ」と「ハンチョウ/20巻」は同じQ番号で関連付けられます。

## 使用例

### 作品へのコメント

```
@@ハンチョウ

宮本さんのセリフ、ぐっときた
```

### 作品の特定巻へのコメント

```
@@ハンチョウ/20巻

宮本さん回、最高だった
```

### 概念への呼びかけ

```
@@酸素

いつもありがとう
```

### URLへの言及

```
@@github.com/nostr-protocol/nips

このリポジトリ、よくまとまってる
```

URLへの言及にはQ番号はつきません。

### 独自の言葉でメンション

```
@@推しの子/アクア

アクアの心情描写が深い
```

Wikidataにない概念や、あえてWikidataと紐付けたくない場合は「新規作成」を選択します。

## 他のNostrクライアントでの表示

スーパーメンションはNostrのタグとして保存されるため、他のクライアントでもタグとして表示されます。

# スーパーメンション

万物に対してコメントできる機能です。

## 概要

通常のメンション `@user` は人への呼びかけ。
スーパーメンション `@@` は**万物への呼びかけ**です。

```
@@ハンチョウ

20巻の宮本さん、最高だった
```

## 構文

```
@@対象
```

- `@@` で始まる（ダブルアットマーク）
- 日本語OK
- スペースは自動でアンダースコアに変換

## 使い方

### 方法1: ボタンから

1. エディタ上部の `@@` ボタンをクリック
2. 検索ボックスに入力
3. 候補から選択

### 方法2: 直接入力

1. エディタで `@@` と入力
2. 対象を続けて入力
3. サジェストから選択（または新規作成）

## サジェスト候補

| アイコン | 意味 |
|----------|------|
| ✓ | 確定済みの履歴（Wikidataと紐付け済み） |
| 📌 | 未確定の履歴 |
| 🔍 | Wikidata検索結果 |
| ✏️ | 新規作成 |

## Wikidataとの連携

候補に表示される `Q番号`（例: `Q12345678`）は Wikidata のIDです。クリックでWikipediaページを開けます。

同じ対象の表記ゆれ（「ハンチョウ」と「1日外出録ハンチョウ」など）も、Q番号で同一視できます。

## 使用例

### 作品へのコメント

```
@@ハンチョウ

宮本さんのセリフ、ぐっときた
```

### 概念への呼びかけ

```
@@酸素

いつもありがとう
```

### URLへの言及

```
@@github.com/nostr-protocol/nips

このリポジトリ、よくまとまってる
```

## 他のNostrクライアントでの表示

スーパーメンションはNostrのタグとして保存されるため、他のクライアントでもタグとして表示されます。

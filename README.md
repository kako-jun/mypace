## 特長

| 機能 | 説明 |
|------|------|
| **分散型** | Nostrプロトコル、特定のサーバーに依存しない |
| **プライバシー重視** | 秘密鍵はローカルのみに保存 |
| **パーソナルカラー** | FF7風4隅グラデーションで自分だけの色 |
| **クリエイティブ** | お絵かき、ボイスメモ、ステッカー |

> 詳細は[ユーザーガイド](./docs/user-guide/index.md)へ

---

## はじめかた

1. [mypace.llll-ll.com](https://mypace.llll-ll.com) を開く
2. 名前を入力（秘密鍵は自動生成）
3. 投稿する

> [クイックスタート](./docs/user-guide/quick-start.md)

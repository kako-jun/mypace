## 特長

| 機能 | 説明 |
|------|------|
| **分散型** | Nostrプロトコル、特定のサーバーに依存しない |
| **プライバシー重視** | 秘密鍵はローカルのみに保存 |
| **パーソナルカラー** | FF7風4隅グラデーションで自分だけの色 |
| **クリエイティブ** | お絵かき、ボイスメモ、ステッカー、画像/動画編集 |
| **ワードロット** | 投稿から単語を収集・合成するゲーム的仕組み |
| **NPC** | AI的なNPCとの共有・対話 |
| **マガジン** | 投稿をまとめて雑誌風に閲覧・編集 |
| **埋め込み** | YouTube、Spotify、TikTok、Twitter、Instagram対応 |
| **スマートフィルター** | 言語・ミュートリスト・プリセットで柔軟にフィルタリング |
| **PWA対応** | インストール可能、プッシュ通知、オフライン対応 |
| **位置情報** | 投稿に位置情報を付加 |

> 詳細は[ユーザーガイド](./docs/user-guide/index.md)へ

## はじめかた

1. [mypace.llll-ll.com](https://mypace.llll-ll.com) を開く
2. 名前を入力（秘密鍵は自動生成）
3. 投稿する
